Self Driving System for Unmanned Ground Vehicle (UGV)

by Yuchen Yao (Personal Project)

Porject Github page

Objective

The objective of this project is to build an unmanned Ground Vehicle that uses machine vision and optical flow to achieve full auto navigation in various environments.

Current Prograss
  1. Designing iteration 2 of the vehicle for better structural stability
  2. Designing and testing different configurations for the dual camera mounting system
  3. Implementing optical flow using 2 cameras.
