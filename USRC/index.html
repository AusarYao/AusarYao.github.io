by Unmanned Aerial Systems at UCLA (UAS@UCLA)

Project Website

Objective

Design and build software and hardware systems that enable drone swarms to lift and transport an object in air for a long period of time.

My Role In the Project

Research, design and implement a mesh network communication system for inter-drone communication and drone-ground communication. In order to achieve this, I combined OONF framework and MOST multicasting algorithm since the control's system is based on ROS2, which uses multicasting for communication.
